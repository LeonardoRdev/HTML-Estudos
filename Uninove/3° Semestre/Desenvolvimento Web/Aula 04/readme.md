### SQL | Structured Query Language
Iremos aprender 4 comandos (CRUD)
- Insert   (C reate)   - Inserir

- Select  (R ead)     - Consultar
SELECT * FROM "sei_la"

- Update (U pdate) - Atualizar / Alterar
